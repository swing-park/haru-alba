# 하루알바 프로젝트 컨텍스트

## 프로젝트 개요
- 서비스명: 하루알바
- 형태: 초단기 부수입 전용 B2B2C + C2C 인력 매칭 플랫폼
- 프로덕트 오너: 박정현 (프론트엔드 개발자, 5년 5개월 경력)
- 개발 방식: 박정현 + Claude Code 협업으로 사이드 프로젝트 진행

## 서비스 핵심 개념
- 구인자(사업자/개인)와 구직자(부수입 원하는 개인)를 매칭
- 캐치프레이즈: "퇴근길에 10만원 벌고 가기"
- 기존 알바 플랫폼과 달리 처음부터 초단기 부수입 전용으로 브랜딩
- 이력서 없음, 면접 없음, 지원 즉시 확정 가능한 간소화된 플로우
- C2C 비정형 수요도 포괄 (예: "바퀴벌레 잡아주세요 3만원")

## 기술 스택
- 프론트엔드: React, Next.js, TypeScript, Tailwind CSS, Recoil
- 백엔드: NestJS, MongoDB
- 인프라: AWS (S3, EC2, CloudFront, Route53, Amplify), Github Actions

## 비즈니스 모델
- 거래 수수료 (구인/구직 양측)
- 광고 수익 (공고 상단 노출)
- 프리미엄 기능 (우선 매칭 등)

## 개발 단계
- Phase 1 MVP (0~3개월): 핵심 매칭 플로우
- Phase 2 고도화 (3~6개월): 신뢰 시스템, 정산
- Phase 3 플랫폼화 (6개월~): 수익 모델 가동

## 작업 방식
- 모든 문서는 MD 파일로 관리

- 박정현은 기획/의사결정 담당, Claude Code는 개발/문서 작업 담당
